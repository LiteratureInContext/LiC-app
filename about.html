Literature in Context

This is a POC for the Literature in Context project in eXist-db.
See more about the project here: https://github.com/tonyahowe/LiC

Included in this POC

Search
Basic and advanced search functions with keyword in context (KWIC) results:
  • Keyword search accross all works
  • Fielded search options for: Title and Author
  • Sort results based on Title, Author, Publication Date, and relevance
  • Add to/Create Coursepack from search results
Browse
Basic browse page for works.
  • Sort on Title, Author and Publication Date
  • Add to/Create Coursepack from browse page
Coursepacks
  • Create new coursepacks
  • Add to existing coursepacks
  • Search within a coursepack
  • Download/Save coursepack in several formats: TEI, PDF, HTML, Text (with more to come)
  • View all available coursepacks
Works
  • View/download works in HTML, PDF, EPUB
  • Page images
Version control/automation from a central repository
I would suggest using github to manage version control and to automatically update your eXist-db instance. I have used this method with a number of github projects and find it to be easy to use with a distributed user group. The code is already written for this, we will need to collaborate to set up the GitHub webhooks access.
